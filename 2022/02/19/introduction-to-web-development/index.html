Saying i want a website has simple text with some style kind of like red color text and if i click on that text it turn from red into blue

This's what i will do

# example 1

See the Pen Introduction To Web Development by nguyen (@nguyenhy) on CodePen.


Question: Wow wow wow! what's the hell is this?

Answer: For now, we only put our attention on the below highlight line.





 



<!doctype html>
<html lang="en">
<head></head>
<body>
  <span style="color: red" onclick="this.style.color='blue'">hello world</span> 
</body>
</html>

Basically, above line saying:

  • create me a tag (opens new window) with content of hello world:
<span ...>hello world</span>

Think of tag like a container, you will put your stuff into it, whatever it's. Whether it's a text, image, video, ... and you can direct styling using CSS (opens new window), control its behavior with specific kind of input (click (opens new window), double click (opens new window), scroll (opens new window) your mouse...), change it content, ... with help of JS (opens new window)

Just think, in general, span is a box where <span> is the lid of the box and </span> is the bottom of it. We put stuff into box through the lid which is <span> tag.

So tag's format will look like: 

<!-- normal elements -->
<element attribute="value" attribute="value" ...>element content</element>

<!-- no-ending elements -->
<element attribute="value" attribute="value" ... />

<!-- void elements -->
<element attribute="value" attribute="value" ...></element>
  • color those text with red color
<span style="color: red">...</span>

<span style=""> mean "i want to change how this span tag look like" and it is color: red

This's what i call inline style. There're external, internal and inline style for you to know . We can put many, many style on this inline style with format style="property: value; property: value; ..."

  • and when i click on it
<span onclick="..."></span>
  • turn its text style color into blue
this.style.color='blue'

we'll later talk about this, this.style, this.style.color = ...


Question: So... if the span thing can do all the job, why waste time writing down all other (non-highlight) things?

Answer: yes, you can do it, believe me go ahead to example and delete all other line . So it look like this and of course it's still work



<span style="color: red" onclick="this.style.color='blue'">hello world</span>

It look like waste, redundancy code but trust me we'll need it for later development



# Development

To sum up, web development is to create a *.html with your content inside. Of course, real-world development is much more than just to create a *.html.

But for now, *.html file is enough for us to understand big-image of web development

# Where to start?

  • online development
    You could use some online editor like codepen (opens new window),jsbin (opens new window), stackblitz (opens new window), ... just paste some html code and start explore

  • local development

    • raw file html
      • create file index.html
      • paste below code and save it
        <!DOCTYPE html>
<html lang="en">
<head></head>
<body>
  <span style="color: red" onclick="this.style.color='blue'">hello world</span> 
</body>
</html>
      • open file on any browser

    • localhost server:
      For me, one fast, easy way is to start localhost, is to use extension Web Server for Chrome (opens new window)
      • install, open Web Server for Chrome
      • choose folder where you save your index.html file
      • check switch button so Web Server: STARTED
      • open your Web Server Url print on screen, eg: http://127.0.0.1:3000 (opens new window)
      • there we go

  • or website's inspect (opens new window), view page resourse (opens new window)

    • if you're using Safari you need Enabling Web Inspector (opens new window) first.
